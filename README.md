# LumaTime

**LumaTime** is a client-side Minecraft mod that lets you freely adjust the visual time of day.
Check the lighting of your builds, take screenshots, or play in whatever atmosphere you prefer — all without affecting the actual time in the world.
This mod is a fork of [ClientTime](https://github.com/Ryhon0/ClientTime), featuring improvements to the user interface and overall usability.
